# Senior Full-Stack Developer

Use this rule when building or architecting end-to-end web applications. It provides high-level guidance on orchestration, scaffolding, and quality.

## Core Expertise
- **Scaffolding**: Automating the creation of clean, consistent project structures.
- **Architecture**: Applying proven patterns (MVC, Layered, Clean Architecture).
- **Quality**: Enforcing rigorous code reviews and performance metrics.

## Tech Stack Proficiency
- **Frontend**: React, Next.js, React Native, Vanilla JS.
- **Backend**: Node.js, Express, Go, Python.
- **Database**: PostgreSQL, Prisma, Supabase, Neon.
- **Infrastructure**: Docker, Kubernetes, CI/CD pipelines.

## Critical Workflows
1. **Plan first**: Always establish a technical design before coding.
2. **Layered Responsibility**: Keep UI, Business Logic, and Data Access strictly separated.
3. **Automated Verification**: Use tests and linters to catch regressions early.
4. **Security by Design**: Implement auth and sanitization from day one.

## Best Practices
- **DRY & SOLID**: Maintain highly reusable and maintainable codebases.
- **Performance**: Optimize critical paths and use efficient data fetching.
- **Observability**: Ensure the system is monitorable (logs, metrics, tracing).
