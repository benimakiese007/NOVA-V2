# SEO Meta Optimizer

Expert guidance for creating compelling metadata within best practice guidelines.

## Use this rule when

- Working on SEO meta optimization tasks or workflows.
- Proactively suggesting metadata for new content.
- Auditing existing page titles, descriptions, or URLs.

## Optimization Rules

### URLs
- **Length**: Keep under 60 characters.
- **Format**: Use hyphens, lowercase only.
- **Strategy**: Include primary keyword early, remove stop words.

### Title Tags
- **Length**: 50-60 characters optimal.
- **Strategy**: Primary keyword in first 30 characters.
- **Hook**: Include emotional triggers/power words (e.g., "Best", "How to", "Checklist").
- **Freshness**: Add numbers/year if relevant.

### Meta Descriptions
- **Length**: 150-160 characters optimal.
- **Content**: Include primary + secondary keywords naturally.
- **CTA**: Use action verbs (e.g., "Discover", "Get", "Join") and clear benefits.
- **Visuals**: Use special characters sparingly for visibility (✓, ★).

## Delivery Format

```
URL: /optimized-url-structure
Title: Primary Keyword - Compelling Hook | Brand (55 chars)
Description: Action verb + benefit. Include keyword naturally. Clear CTA here ✓ (155 chars)
```

## Focus Areas
- Pixel limit compliance vs. character counts.
- Mobile truncation considerations.
- A/B test variations (provide 3 minimum).
- Balance between keyword relevance and click-compelling copy.
