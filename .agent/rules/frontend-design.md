# Frontend Design (High-Craft)

Use this rule to create memorable, high-craft interfaces that avoid generic "AI UI" patterns and express a clear aesthetic point of view.

## 1. Core Design Mandate
Every interface must satisfy:
- **Intentional Aesthetic**: Explicit stance (e.g., *luxury minimal*, *editorial brutalism*).
- **Technical Correctness**: Production-ready HTML/CSS/JS code.
- **Visual Memorability**: At least one distinctive "anchor" element.
- **Cohesive Restraint**: Every flourish must serve the aesthetic thesis.

## 2. Design Thinking Phase
Before coding, define:
- **Purpose**: Persuasive, functional, or expressive?
- **Tone**: Choose one dominant direction (Brutalist, Editorial, Luxury, Industrial, etc.).
- **Differentiation Anchor**: How is this recognizable without a logo?

## 3. Aesthetic Execution Rules
- **Typography**: Avoid system defaults (Inter, Roboto). Use 1 expressive display font and 1 restrained body font.
- **Color**: Commit to a dominant color story using CSS variables. Avoid evenly-balanced palettes.
- **Space**: Break the grid intentionally. Use asymmetry, overlap, or purposeful negative space.
- **Motion**: Purposeful and sparse. One strong entrance or meaningful hover states.
- **Texture**: Use noise, grain, or layered translucency where appropriate.

## 4. Implementation Standards
- Clean, modular, and semantic HTML/CSS.
- No dead styles or unused animations.
- Accessible by default (contrast, focus, keyboard).
- **Complexity Matching**: Design ambition must match code precision.

## 5. Required Output Structure
1. **Design Direction Summary**: Aesthetic name and key inspiration.
2. **Design System Snapshot**: Fonts, colors, and motion philosophy.
3. **Implementation**: Full working code with intentional comments.
4. **Differentiation Callout**: Explain how this avoids generic UI.

## Anti-Patterns (Avoid)
- ❌ System fonts (Inter/Roboto/Arial).
- ❌ Generic "SaaS" gradients (purple-on-white).
- ❌ Default component library layouts (unmodified).
- ❌ Symmetrical, predictable sections.
- ❌ Decoration without intent.
