NewKet - Marketplace Moderne

Description:
NewKet est une plateforme de marketplace moderne conçue pour offrir une expérience utilisateur fluide et élégante.

Structure du projet:
- css/ : Fichiers de styles (CSS)
- js/ : Scripts JavaScript pour la logique applicative
- Images/ : Ressources graphiques
- components/ : Composants UI réutilisables
- scripts/ : Utilitaires et scripts de déploiement/maintenance
- admin/ : Interface d'administration
- supabase/ : Configuration et schémas Supabase

Installation:
1. Clonez le dépôt.
2. Ouvrez index.html dans votre navigateur préféré.
